Process View

Clicking on a JDF node in the Tree View will draw the Process View for that Node. The In- & Output View is automatically selected; to view the Processes click on the Process View Tab. From here it is possible to navigate the Processes by doing a mouse click with the left mouse button on a JDF Node; the processes that Node describes is then displayed. A click with the left mouse button on a Resource will select that Resource in the Tree View.
