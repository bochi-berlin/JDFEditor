Building a .jar File

When building a .jar File a couple of changes has to made in certain classes. The affected classes, and the number of changes that has to be made in them, is the following:

  • JDFFrame - Two Changes

  • HelpFrame - Two Changes

  • PreferenceDialog - One Change

  • INIReader - One Change

The changes to be made is to Comment and Uncomment a number of Lines in the code that initiates the ImageIcons. The reason for this is so that the GIF Files can be included in the .jar File.

Include the following Resources when exporting a .jar File:

  • JDFEditor/src

  • JDFEditor/icons

  • JDFLib/src

  • Xerces/src
