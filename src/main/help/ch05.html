Suggentions of Further Development

The following features have not been implemented in the CIP4 JDF Editor and this is a suggestion of features to add to the application;

  • Schema Validation

  • Undo and Redo

  • Source View

  • Improve Validation after Editing

  • Enhance JMF Support

  • Change Comment-text directly in the Comment View

One other possible feature to add is the ability to execute the JDF process associated with the JDF file displayed in the editor with the return URL setup to notify the editor so that it could be automatically reloaded after process execution. This would allow the user to see how process execution modifies the JDF and allow a quick means for testing processes.

Yet another feature is to implement the ability to put together a table with starting point in a JDF file. The table elements will contain information about all the possible processes, resources and attributes in the JDF file. This table will be useful for interoperability purposes.
