Chapter 3. Edit

Editing is possible from the Tree View. The editing Options for the selected Node is available in a Popup Menu, which is displayed when doing a mouse click with the right mouse button, and also in the Edit and Insert Menues. When inserting a new element or attribute, only valid options are eligible.

In the Preferences Options, which are found in the Tools Menu, you can choose to Open a File Read Only under the General Tab. That means that the File can not be edited if it was Opened with the Read Only Option selected. After Opening a File it is not possible to change the editing mode. To enable editing the File has to be closed, the Read Only Option turned off and the File has to be opened again.

The Undo and Redo features are not yet supported...
