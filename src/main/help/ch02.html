Tree View

The Tree View displays the JDF or JMF File as a Tree. Both the elements and the attributes are displayed as Tree Nodes. If the automatic validation is on or the Validate Button is pressed, invalid elements and attributes are displayed with the Error Icon. A number of different icons are displayed in the tree view.
